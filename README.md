# RoBERTa-SC: LLM-Powered Semantic Communication via Efficient Signal-Text Alignment

This repository contains the code for the paper "RoBERTa-SC: LLM-Powered Semantic Communication via Efficient Signal-Text Alignment", currently under review.

To protect the intellectual property during the peer-review process, the source code is temporarily kept private. We are committed to open science and will release the full codebase and pre-trained models immediately upon the acceptance of the paper.

We appreciate your understanding and interest in our work. Please check back later or watch this repository for updates.
